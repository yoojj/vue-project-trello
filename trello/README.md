# Vue Trello

```bash
$ vue create trello
$ vue add vuex router axios i18n

$ npm install

$ npm run serve
```


**구조**  

```bash
src/
├── _constant/       # 상수
├── _plugins/        # 전역 메소드
├── _utils/          # 유틸
├── _vueI18n/        # 국제화
├── _vuex/           # 상태 관리  
├── assets/          # 정적 자원
├── components/      # vue 컴포넌트  
├── layout/          # vue 페이지 레이아웃
├── view/            # vue 페이지
├── App.vue         
├── i18n.js
├── main.js
└── router.js
```
