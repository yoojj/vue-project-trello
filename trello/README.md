# Vue Trello

```bash
$ yarn install @vue/cli

$ vue create trello

$ vue add vuex
$ vue add router
$ vue add axios
$ vue add i18n

$ yarn install

$ yarn run serve
$ yarn run build
```


**구조**  

```
src/
├── _constant/       # 상수
├── _plugins/        # 전역 메소드
├── _utils/          # 유틸
├── _vueI18n/        # 국제화
├── _vuex/           # 상태 관리  
├── assets/          # 정적 자원
├── components/      # vue 컴포넌트  
├── layout/          # vue 페이지 레이아웃
├── view/            # vue 페이지
├── App.vue         
├── i18n.js
├── main.js
└── router.js
```
